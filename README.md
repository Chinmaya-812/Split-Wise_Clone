# Splitwise Clone Backend

**About:**  
Node.js backend for a Splitwise-like expense sharing application. Allows users to create accounts, manage shared expenses, calculate balances, and track group debts through a secure, RESTful API.

## Features

- User authentication with JWT
- Create and manage users, groups, and expenses
- Calculate balances and who owes whom
- RESTful API endpoints for all operations
- Modular and scalable backend structure
- Optional: Notifications for pending payments

## Technologies Used

- Node.js and Express.js
- MongoDB or PostgreSQL for data storage
- JWT for authentication
- RESTful API design
